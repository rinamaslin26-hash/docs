This page contains information about the games various cheats and how to enter them.

# Entering Cheat Codes
## On PC
Hold F1 while in the top level of the Settings menu and enter the code using the arrow keys.

On the main menu, doing this will crash the game unless you have the **Main Menu Cheat Code Enter** setting enabled in the [[/LucasSimpsonsHitAndRunModLauncher/Hacks/BugFixes.md]] hack.

## On GameCube / Xbox
Hold L+R in the top level of the Settings menu and enter the code using the face buttons.

## On PlayStation 2
Hold L1+R1 in the top level of the Settings menu and enter the code using the face buttons.
