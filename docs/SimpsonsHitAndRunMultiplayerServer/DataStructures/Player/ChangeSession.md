Moves the player to another session. While this method was added in version 1.0, it is not recommended for use yet.

# Syntax
```lua
Player:ChangeSession( session )
```
 
## Arguments
* **session** ([[../../LuaScripting/Session.md|Session]]): The session to move the player to.

## Return Values
No return values.
