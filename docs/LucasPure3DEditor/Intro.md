Lucas' Pure3D Editor is a tool for opening Pure3D (.p3d) files found in various games created by Radical Entertainment.

The tool primarily focuses on support for Pure3D files and chunks found in The Simpsons: Hit & Run with limited support for chunk types found in other games.
