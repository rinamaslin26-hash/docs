# New Mods 
## Unlock All Rewards
Added this new mod. This is a mod that simply enables the new **Unlock All Outfits**, **Unlock All Vehicles** and **Unlock Bonus Game** hacks.

# New Hacks
## Custom Bonus Mission Support
Added this new hack. This allows you to specify custom street race characters.

## Custom Limits
Added this new hack. This allows mods to customise various limits to suit their needs.

## Unlock All Missions
Added this new hack. This unlocks all levels and missions when enabled.

## Unlock All Outfits
Added this new hack. This hack has to be required by a mod.

## Unlock All Vehicles
Added this new hack. This hack has to be required by a mod.

## Unlock Bonus Game
Added this new hack. This hack has to be required by a mod.
