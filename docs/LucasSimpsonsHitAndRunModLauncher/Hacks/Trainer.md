This hack adds an ingame window for tweaking game parameters on the fly, teleporting around, and more.

# Settings
## Keybinds
### Toggle Trainer Visibility
Toggles the visibility of the trainer window.

**Defaults to L.**

## Configuration
### Setting Step Size
Sets the amount -/+ buttons will change a value when clicked once.

**Defaults to 0.1.**

### Setting Step Size - Held
Sets the amount -/+ buttons will change a value when held down.

**Defaults to 1.**

## Input
### Disable Game Input When Trainer Is Open
Disables input to the game when the trainer window is open.

**Defaults to Enabled.**

### Disable Hack Input When Trainer Is Open
Disables input to other hacks when the trainer window is open.

**Defaults to Enabled.**
