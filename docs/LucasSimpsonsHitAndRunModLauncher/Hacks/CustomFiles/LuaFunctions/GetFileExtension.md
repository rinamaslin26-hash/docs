Returns the file extension of the specified path.

# Syntax
```lua
GetFileExtension( <path> )
```

* **path**: The path or filename of the file you want to get the extension of.

# Examples
```lua
-- Returns ".mfk"
GetFileExtension("example.mfk")
```
