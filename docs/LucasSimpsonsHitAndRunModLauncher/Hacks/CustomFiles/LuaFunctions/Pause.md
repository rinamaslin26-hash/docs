Pauses the console and prompts the user to "Press any key to continue...".

# Syntax
```lua
Pause()
```

# Examples
```lua
Pause() -- Does as described on the tin.
```

# Notes
If the [[../../Console.md]] hack or its **Console > Enabled** setting is not enabled, a console window will still be shown temporarily to prompt the user.
