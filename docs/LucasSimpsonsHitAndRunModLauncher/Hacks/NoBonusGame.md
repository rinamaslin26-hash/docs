This hack adds disables the bonus game.

# Requiring This Hack
To require this hack, add this line to your mod's Meta.ini:

```ini
RequiredHack=NoBonusGame
```

# Notes
The game will still imply that the player unlocked a bonus game track when collecting every card in a level or when collecting every card in the game.

It is up to a mod using this hack to modify or blank out this message however the mod sees fit.
