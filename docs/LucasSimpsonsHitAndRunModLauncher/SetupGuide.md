This page is a guide on how to download and setup the Mod Launcher.

# 1: Downloading the Mod Launcher
First things first, you will want to download the latest version of the Mod Launcher from it's Mod Bakery page:
[Project:6]

![A screenshot showing mouse cursor hovering over the Mod Launcher's download button on Mod Bakery](/img/LucasSimpsonsHitAndRunModLauncher/SetupGuide/DownloadButton.png)

# 2: Extracting the Mod Launcher
Next, you want to extract the ZIP archive of the Mod Launcher you just downloaded.

## Using Windows
On Windows, you can extract the archive using 
To extract the archive using the built in Windows utility, simply **Right Click** the ZIP file and choose "Extract All...** to open the **Extract Compressed (Zipped) Folders dialog**.

From there, choose where you want to extract the Mod Launcher and then click **Extract**.

## Using 7-Zip
To extract the archive using [7-Zip](https://www.7-zip.org/), simply **Right Click** the ZIP file and choose **7-Zip > Extract files...**

From there, choose where you want to extract the Mod Launcher and then click OK.

# 3: The Mod Launcher's Folder
Now, you should have a folder containing a few things:

* A folder named **DLLs**: This folder contains DLLs and other files used by the Mod Launcher.
	* Unless explicitly instructed to do so by Donut Team, you should avoid making changes to the contents of this folder.
* A folder named **Mods**: This is one of several places where you can install mods for the Mod Launcher. There are also a few small mods included in this folder.
* **Lucas Simpsons Hit & Run Mod Launcher.exe**: This is, of course, the actual launcher itself.
* **Lucas Simpsons Hit & Run Mod Launcher.exe.config**: This is a .NET configuration file for the Mod Launcher's executable.

# 4: First Time Setup
Let's go ahead and start up the Mod Launcher.

Upon first startup, you will be prompted to sign into your Donut Team account to enable [[DonutTeamAccountIntegration.md]].

![A screenshot of the Mod Launcher's account window](/img/LucasSimpsonsHitAndRunModLauncher/SetupGuide/AccountWindow.png)

This is **Fully Optional** and you may simply press **Skip** if you do not wish to do so.

# 5: Launching the Game
After the account prompts, you'll be greeted with the Main Window of the Mod Launcher which should look like this:

![A screenshot of the Mod Launcher's main window](/img/LucasSimpsonsHitAndRunModLauncher/SetupGuide/MainWindow.png)

From here, you can simply click the **Launch** button on the bottom right to launch the game.

The first time you do this, you will be prompted to browse for your install of the game. Where the game is installed to can vary but the default installion locations are:
* **32-bit Systems**: `C:\Program Files\Vivendi Universal Games\The Simpsons Hit & Run`
* **64-bit Systems**: `C:\Program Files (x86)\Vivendi Universal Games\The Simpsons Hit & Run`

Once you navigate to your install, select **Simpsons.exe** and click **Open**.

Once you've done that, the game will launch and you are done setting up the Mod Launcher!
